Na aula 5 foi visto os seguintes tópicos.

  • Comentários
    • // - usado para comentar uma única linha
    • /* bloco de texto */ - usado para comentar todo o bloco de texto que estiver entre /* e */
  • Variáveis / Tipos primitivos
    • string: 'Wagner', '30'
    • boolean: true, false
    • number: 1, 3.75, 0.123, 99
      • Infinity
      • NaN (Not a Number)
    • null
    • undefined
    • object
      • Array
    • function

O comando typeof pode ser usado para verificar qual o tipo primitivo da variável/objeto em questão:

> typeof 'Wagner'

Este comando irá retornar 'string'.
