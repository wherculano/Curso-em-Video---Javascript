Operadores:
  • Aritméticos
  • Atribuição
  • Relacionais
  • Lógicos
  • Ternário
Aritméticos
  • + Adição
  • - Subtração
  • * Multiplicação
  • / Divisão
  • % Módulo (Resto da divisão)
  • ** Potenciação
Ordem de Precedência
  • ( )
  • **
  • / * %
  • + -
Atribuição Simples:
  • var a = 10
Auto-atribuição
  • var n = 50
  • n += 5
  • n -= 1
  • n *= 2
  • n /= 6
  • n %= 4
  • n **= 3
    • No fim de tudo, n valerá 8
Incremento/Decremento
  • var n = 10
  • n++ (pós incremento: após a operação n será incrementado de 1 (n +=1))
  • ++n (pré incremento: antes a operação n será incrementado de 1 (n +=1))
  • n-- (pós decremento: após a operação n será decrementado de 1 (n -=1))
  • --n (pré decremento: antes a operação n será decrementado de 1 (n -=1))
