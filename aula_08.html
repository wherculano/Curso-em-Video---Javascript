Operadores:
  • Aritméticos
  • Atribuição
  • Relacionais
  • Lógicos
  • Ternário
Relacionais
  • > Maior
  • < Menor
  • >= Maior ou Igual
  • <= Menor ou Igual
  • == Igual (Resto da divisão)
  • != Diferente

Os resultados utilizando operadores relacionais serão sempre um booleano

  • 5 > 2 .. true
  • 7 < 4 .. false
  • 8 >= 8 .. true
  • 9 <= 7 .. false
  • 5 == 5 .. true
  • 4 != 4 .. false
Identidade (comparando 5 Number com '5' String):
  • 5 == 5 .. true
  • 5 == '5' .. true
    • Neste caso o JS analiza se os valores são o mesmo (não o tipo).
  • 5 === 5 .. true
  • 5 === '5' .. false
    • Neste caso o JS analiza se os valores E se os tipos são os mesmos.
  • 5 != '5' .. false
  • 5 !== '5' .. true
    • O segundo caso é chamado de Desigual Restrito. Pois os tipos são diferentes mas os valores são os mesmos.
Lógicos
  • ! (negação/não)
    • !true = false
    • !false = true
  • && (conjunção/e)
    • true && true .. true
    • true && false .. false
    • false &&true .. false
    • false &&false .. false
  • || (disjunção/ou)
    • true || true .. true
    • true || false .. true
    • false ||true .. true
    • false ||false .. false
Ordem de Precedência: Aritméticos, Relacionais, Lógicos
  • 5 > 8 && 8 % 2 == 0
  • (5 > 8) && ( (8 % 2) == 0)
  • (5 > 8) && (( 0 ) == 0)
  • false && true
  • false
  • 5 <= 8 || 8 / 2 ==2
  • (5 <= 8) || ( (8 / 2) == 0)
  • (5 <= 8) || (( 4 ) == 0)
  • true || false
  • true
    • A ordem de Precedência será sempre: !, &&, ||
Ordem de Precedência "Geral"
  • Aritméticos: (), **, /, * , %
  • Relacionais: >, <, >=, <=, ==, !=
  • Lógicos: !, &&, ||
